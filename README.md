This project builds a machine learning pipeline to detect fraudulent credit card transactions using two classification algorithms — Decision Tree and Support Vector Machine (SVM). 

The goal is to correctly identify fraudulent transactions while minimizing false negatives, which represent real fraud cases that go undetected.

The dataset used is the real-world Kaggle Credit Card Fraud Detection dataset, containing transactions made by European cardholders in September 2013.


Using ML algorithims like desicion trees and SVM's, we have made an attempt to identify fraudalent transactions by testing on small scale data- upto 90% accuracy.

To be able to use this code to it's fullest accuracy, make sure to download the kaggle dataset https://www.kaggle.com/mlg-ulb/creditcardfraud
